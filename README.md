# WAD-Lab11


## Pre-requisites
Node.js \& NPM

## Clone the repository
```
git clone https://github.com/M-Gharib/WAD-Lab11.git
```

## Navigate to the project directory
```
cd <project name>
```

## Install dependencies
```
npm install
```

## Run the project
```
node server
```

or 
```
nodemon server
```
